What We Do!

The Urban tree management is a system which provides where to plant, to search seeds and fertilizer easily online.
We help you with where to plant trees in your nearby location with the help of map visualization.

Our Goal!

Register Yourself

Registered users can search for bio-fertilizers, where to plant trees in their nearby location, track the growth of their plantation with unique tree ID. Also tree manager who caretake the user's tree growth can interact with the website by uploading latest plant condition so users can see the current condition of their plant.

Map Visualization

With the help of maps user can select their nearby location for plantation. Using google maps API, tree data of order plantation will be visualized on the map. Users get a unique tree ID, After entering this ID it shows each user the precise location of the tree they have planted, who is taking care of it, and much more.

Services

Search Bio-Fertilizers

We recommend bio-fertilizer for best health of plant and promote reduce, reuse and recycle. We provide optimized search option in your neartby location. We provide verified trusted sellers and quality assurances of the product.

Visualized Map Data

We provide map visualization to display users planted tree geographically and present it in the form of maps data. This map can display a large amount of heirarchical data. Using unique tree id, users can track growth of their tree data by entering ID.

Unique Tree ID

We provide 6 digit unique tree ID which users get upon ordering or planting tree at respected location. This ID can also be used for creating query tickets and many more. We record all the information of each and every planted trees.
